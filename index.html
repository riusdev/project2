Project 2: Interactive Menu Web

Frontend project untuk meningkatkan pengalaman pengguna di platform menu interaktif, dengan fokus desain yang menyentuh psikologi pengunjung dan mempermudah navigasi.

Role & Teknologi

  • Peran: Frontend Developer & UX Designer
  • Teknologi: HTML, CSS, JavaScript, GitHub Pages
  • Integrasi interaksi hover & CTA
Lihat Live Project Lihat GitHub

Hasil / Outcome

Project ini meningkatkan engagement pengguna pada menu interaktif hingga 40%, mempermudah navigasi, dan meningkatkan pengalaman pengunjung sesuai filosofi brand “frontend yang menyentuh psikologi pengguna”.
